Find On Git

FIND ON GIT is a web app that allows users to search GitHub profiles and view detailed information and repositories in real-time. Built with HTML,CSS, and JavaScript, it uses Axios to fetch data from the GitHub API asynchronously. The app provides seamless profile searches, displaying relevant details and handling errors like unfound profiles or repository issues to enhance the user experience.
